# Свадебное приглашение — сайт

Красивое свадебное приглашение в виде одностраничного сайта. Работает без сервера — просто откройте `index.html` в браузере.

---

## Структура файлов

```
wed_invite/
├── index.html   — разметка
├── style.css    — стили
├── script.js    — анимации, таймер, эффекты
└── README.md    — эта инструкция
```

---

## Как открыть локально

Просто откройте файл `index.html` в любом современном браузере (Chrome, Safari, Firefox, Edge).
Сервер не нужен.

---

## Как заменить данные

Откройте `index.html` в любом текстовом редакторе и найдите следующие места:

| Что менять | Где искать |
|---|---|
| Имена пары | `Александр &amp; Мария` (несколько мест) |
| Дата свадьбы | `14 июня 2025` |
| Адрес площадки | `г. Москва, Рублёвское шоссе, д. 15` |
| Название площадки | `Усадьба «Белый сад»` |
| Текст истории | Раздел `<!-- OUR STORY -->` |
| Время в программе | Раздел `<!-- SCHEDULE -->` |

В файле `script.js` строка 2 (таймер обратного отсчёта):
```js
const WEDDING = new Date('2025-06-14T15:00:00');
```
Замените дату и время на ваши.

### Добавить реальные фотографии

В разделе `<!-- GALLERY -->` каждый блок `.gallery__placeholder` замените на тег `<img>`:
```html
<!-- было -->
<div class="gallery__placeholder gallery__placeholder--1">
  <span class="gallery__text">фото скоро</span>
</div>

<!-- стало -->
<img src="photos/photo1.jpg" alt="Александр и Мария" />
```

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub (например `wedding`).
2. Загрузите все файлы из папки `wed_invite/` в корень репозитория.
3. Перейдите: **Settings → Pages → Source → Deploy from a branch → main / (root)**.
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://<ваш-логин>.github.io/wedding/`

### Быстрый способ через Git

```bash
cd путь/к/wed_invite
git init
git add .
git commit -m "Wedding invitation site"
git branch -M main
git remote add origin https://github.com/<ваш-логин>/wedding.git
git push -u origin main
```

Затем включите GitHub Pages в настройках репозитория (см. выше).

## Кастомизация цветов

Все цвета заданы в CSS-переменных в начале `style.css`:

```css
--blush:  #f8e8e0;   /* нежно-розовый фон */
--cream:  #fdf6f0;   /* кремовый фон секций */
--rose:   #d4a5a5;   /* пыльная роза */
--gold:   #c9a96e;   /* золото */
```
